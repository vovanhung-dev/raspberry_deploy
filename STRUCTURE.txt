Features:
=========
✓ Upload image prediction
✓ Real-time camera detection
✓ 29 ASL sign classes (A-Z, del, nothing, space)
✓ Top-5 predictions with confidence
✓ FPS counter
✓ Responsive UI
✓ Auto-install script
✓ Run script

Installation Size After Setup:
================================
- Minimal install: ~550 MB (with TFLite)
- Full install: ~900 MB (with TensorFlow)

Performance on Raspberry Pi 5:
===============================
- Image prediction: 1-3 seconds
- Camera FPS: 2-5 FPS
- RAM usage: 500 MB - 1 GB
- Model load time: 5-10 seconds

Quick Start:
============
1. Copy folder to Raspberry Pi
2. Run: ./install.sh
3. Run: ./run.sh
4. Access: http://<pi-ip>:5000

See QUICKSTART.md for detailed instructions.
